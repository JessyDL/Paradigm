Detailed Description

does not contain any actual texture data, just describes it. This aids the engine in setting up the correct resources for rendering the associated texture file.

Warning
The description is assumed to be correct, otherwise undefined behaviour happens.
Note
although this class allows runtime editing, it's unlikely you'd need it unless you generate textures. and wish to serialize them to disk.
